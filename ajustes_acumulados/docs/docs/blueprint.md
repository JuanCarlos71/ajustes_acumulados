# **App Name**: Inventory Insights Analyzer

## Core Features:

- XLSX File Upload: Upload two XLSX files: one containing warehouse adjustment movements and the other with inventory valuation factors.
- Data Analysis: Analyze the uploaded data to identify discrepancies between warehouse adjustments and system inventory values, categorize them by movement type and center.
- Discrepancy Prioritization: Identify products with the largest discrepancies, sorted from smallest to largest difference in valuation.
- Automated Insight Generation: Leverage AI to generate textual insights on the causes of the differences.
- Downloadable Report: Provide a button to download the analysis results as an XLSX file.
- Data Visualization: Visualize key metrics using charts, such as a pie chart showing the distribution of inventory differences across product categories.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey professionalism and precision.
- Background color: Light teal (#E0F8F8), a desaturated tint of the primary color, for a clean and subtle backdrop.
- Accent color: Yellow-gold (#B8860B), an analogous color to deep teal that creates emphasis on key insights and actions.
- Body font: 'Inter', a sans-serif font for a modern and easily readable text. Headline font: 'Space Grotesk' a slightly more attention-grabbing sans-serif used for headlines, with 'Inter' for body.
- Use clean, simple icons to represent movement types and data categories.
- Implement a clear, tabular layout for presenting the analysis results, making it easy to compare data across different dimensions.
- Use subtle transitions and loading animations to provide feedback during data processing and report generation.
